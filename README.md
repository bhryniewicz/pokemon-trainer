## Getting Started

Install all dependencies

```bash
npm i
```

if you are using lower version of Node thab v18.17.0 then:

- you need to have a nvm to change version

```bash
nvm use v18.17.0
```

TO start development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

To do a production build:

```bash
npm run build => npm run start

```

To check tests

```bash
npm run test
```
